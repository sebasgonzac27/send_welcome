# Send Welcome Email Application
![image](/assets/preview.png)

## Steps to use

1. Create environment file `.env` with following variables.
```
MAIL_HOST=
MAIL_PORT=
MAIL_AUTH_USER=
MAIL_AUTH_PASS=
MAIL_FROM_ADDRESS=
COACH_PHONE=
COACH_NAME=
```
2. Install the dependencies `npm install`.
3. Execute the application `npm run start`.
